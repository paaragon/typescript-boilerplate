# Typescript Boilerplate

Typescript boilerplate with test, coverage, lint, logger and config by environment.

## Commands

- `npm build`: transpile typescript (only src).
- `npm start`: `npm run build` + execute build with node.
- `npm run start:dev`: execute `src/index.ts` with ts-node-dev and respawn on code changes.
- `npm run lint`: lints `src/**/.ts`.
- `npm run lint:fix`: fix `src/**/*.ts`.
- `test`: test everything under `test` like `*.test.ts` and gives coverage report.
